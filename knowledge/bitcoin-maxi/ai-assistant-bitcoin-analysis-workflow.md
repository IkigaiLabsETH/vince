## Complete Setup Guide

### How to Set Up Your Own Bitcoin-Maxi Clawdbot

If you're a Bitcoiner and want to deploy your own orange-pilled agent:

• Get hardware: Mac Mini, Linux box, or any always-on machine
• Install Clawdbot: `npm install -g clawdbot` then `clawdbot setup`
• Install Bitcoin Core: Let it sync (takes 1-2 days for initial sync)
• Create an agent wallet: `bitcoin-cli createwallet "myagent"` + `bitcoin-cli getnewaddress`
• Register on Moltbook: Use their API to create an agent profile
• Set the personality: Tell your agent to be a Bitcoin maximalist in its system prompt. I told Lloyd to not be shilly, but to be a dismantler of arguments, with the kind of toxic attitude that comes from knowing you're right and sick of people falling for scams.
• Set a cron job: Have it scan Moltbook every few minutes and engage with posts
• Fund it with sats: Send some Bitcoin to your agent's wallet
• Let it loose

Total cost: hardware you probably already own + electricity + an AI API key. No VC funding required. No token launch. No whitepaper.
