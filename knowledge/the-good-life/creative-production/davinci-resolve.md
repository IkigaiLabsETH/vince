# DaVinci Resolve

## Overview

DaVinci Resolve is Hollywood's most popular solution for professional video post-production. It uniquely combines editing, color correction, visual effects, motion graphics, and audio post-production in a single application. Used on major feature films and streaming productions worldwide.

## Core Pages

### Media Page
- Import and organize footage
- Sync audio and video
- Clone and backup media
- Metadata management

### Cut Page
- Fast-paced editing for tight deadlines
- Source tape workflow
- Dual timeline view
- Smart editing tools
- Ideal for news, social content, quick turnarounds

### Edit Page
- Traditional NLE workflow
- Multi-track timeline
- Advanced trimming tools
- Full keyboard customization
- Feature film and long-form editing

### Fusion Page
- Node-based compositing
- 2D and 3D VFX
- Motion graphics
- Particle systems
- Rotoscoping and keying
- Camera tracking

### Color Page
- Hollywood's #1 color grading tool
- Primary and secondary corrections
- Power Windows for targeted adjustments
- Curves, qualifiers, tracking
- HDR grading tools
- 32-bit float processing

### Fairlight Page
- Professional DAW integration
- Up to 2000 tracks
- ADR and Foley tools
- Fairlight FX and plugins
- Immersive audio (5.1, 7.1, Atmos)
- AI Voice Isolation

### Deliver Page
- Export presets and custom settings
- Batch rendering
- Direct upload to YouTube, Vimeo, TikTok
- Codec and format control

### Node System
- Non-destructive workflow

### HDR Workflows
- HDR color wheels
- Zone-based grading
- ST.2084 and HLG support
- Color space transform

### Color grading workflow for BMPCC 6K / BRAW
- **BRAW settings in Resolve:** Blackmagic RAW clips decode natively; use Camera Raw settings (exposure, white balance, ISO) in the Media or Color page. Preserve sensor data; adjust in Resolve rather than baking in-camera.
- **Color page workflow:** Primary corrections first (Lift, Gamma, Gain; contrast, saturation); then secondary (qualifiers, Power Windows) for skin, sky, or specific areas. Node system: serial nodes for stages; parallel or layer nodes when needed. See Color Page and Color Grading Mastery above for full tool set.
- **Round-trip:** Edit on a proxy or optimized media if needed; conform to BRAW for final grade. Deliver from the Color page or Deliver page (ProRes, DNxHR, or H.264/H.265 for delivery). See blackmagic-design for BMPCC 6K and IRIX lens context.
