# Ableton Push 3

## Overview

Ableton Push 3 is a revolutionary music-making instrument that can operate as either a standalone device or as a controller for Ableton Live. It represents the pinnacle of hardware/software integration for electronic music production and performance.

## Key Capabilities

### Standalone Mode
- Built-in Intel Core i3 processor with 8GB RAM
- 256GB SSD storage
- 2.5 hours battery life
- Complete independence from computer for rehearsal and performance
- Seamless project transfer to/from Ableton Live

### As Live Controller
- Deep integration with Ableton Live
- Real-time control of every musical parameter
- Clip launching and arrangement
- Device and mixer control

## Expressive Playing

### MPE-Enabled Pads
- 64 velocity and pressure-sensitive pads
- Per-pad pitch bend, slide, and pressure
- Polyphonic aftertouch for nuanced expression
- Gestures become part of the performance

### Touch-Sensitive Step Editing
- Bring melodies, harmonies, and rhythms to life
- Intuitive sequencing workflow
- Real-time recording and editing

## Sampling Capabilities

### Built-in Sampler
- Load audio files or sample from external sources
- Warp, slice, and reshape audio
- Stem Separation (Live 12.3+ Suite) for isolating vocals, drums, bass
- Map slices to pads for tactile performance

### Recording Studio Features
- Built-in audio interface with dual inputs
- Balanced outputs for studio monitors
- CV/gate outputs for modular gear
- MIDI ports for external hardware

## Sound Design

### Synthesizer Control
- Direct access to Wavetable, Drift, Meld instruments
- Full-screen parameter visualization
- Hands-on control of oscillators, filters, modulation
- Real-time sound shaping

## Performance Features

### Live Show Ready
- Robust build for touring
- Clear visual feedback with high-contrast display
- Real-time vocal and instrument capture
- Effect manipulation during performance

## Creative Workflow Benefits

1. **Disconnect to Create**: Work without computer distractions
2. **Tactile Expression**: Physical interaction with music
3. **Seamless Integration**: Move between standalone and Live
4. **Professional Quality**: Studio-grade audio interface built-in
5. **Modular Ready**: CV/gate for hardware synthesizer integration

## Learning Path Considerations

- Start with basic pad playing and clip launching
- Progress to step sequencing and melodic patterns
- Explore sampling and audio manipulation
- Advance to live performance and arrangement
- Master integration with modular and external gear

## Ideal Use Cases

- Live electronic performance
- Beat making and production
- Sound design sessions
- Sampling and remixing
- Portable music creation
- Integration with modular synthesizers

## House music production and AI/MCP

**Typical MCP use cases:** Control tempo, clip launch, transport; tweak effects and device parameters; automate arrangement or mixing from outside Live. An MCP server can send OSC or MIDI to Live so Claude (or another client) drives the DAW from natural language.

**Exposing Live via OSC or Max for Live:** Ableton Live supports OSC (Open Sound Control) for remote control. Max for Live can expose custom endpoints or bridge to other APIs. An MCP server can run locally and translate tool calls into OSC or Max messagesâ€”e.g. "set tempo 128", "launch clip 2 on track 1", "toggle device on".

**Push 3 standalone vs controller for production:** Standalone = no computer; sessions live on the device; good for sketching and performance. Controller = Live on computer; Push 3 as surface; full plugins and MCP/OSC from the host. For house music production with AI/MCP, controller mode is typical so the MCP server and Live run on the same machine.
