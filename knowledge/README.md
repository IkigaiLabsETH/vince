# Knowledge Base

The knowledge base provides **thinking frameworks and methodologies**, not current data. Agents should use it for _how to think_ about topics—analytical approaches, pattern recognition, strategic frameworks—and fetch live data via actions/APIs.

See [KNOWLEDGE-USAGE-GUIDELINES.md](KNOWLEDGE-USAGE-GUIDELINES.md) for usage rules and agent implementation.
