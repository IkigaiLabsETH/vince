Each index file lists:

1. **Primary categories** — why they matter for the agent
2. **Key files** — most relevant files with one-line descriptions
3. **Data sources** — cron jobs and feeds that power the agent

Use these indexes to route queries to the right agent and ensure each agent loads the most relevant context.
