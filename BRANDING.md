### Color Palette (Product / App)

Defined in `src/frontend/index.css` (OKLCH). Dark mode is default.

| Role | Light | Dark (default) | Usage |
|------|--------|-----------------|--------|
| **Background** | `oklch(1 0 0)` | `oklch(0.2029 0.0037 345.62)` | Page, panels, cards |
| **Foreground** | `oklch(0.145 0 0)` | `oklch(0.9851 0 0)` | Primary text |
| **Primary** | `oklch(0.205 0 0)` | `oklch(0.4703 0.2364 263.19)` | CTAs, links, key UI |
| **Primary foreground** | `oklch(0.985 0 0)` | `oklch(0.9851 0 0)` | Text on primary |
| **Muted** | `oklch(0.97 0 0)` | `oklch(0.2393 0 0)` | Secondary surfaces |
| **Muted foreground** | `oklch(0.556 0 0)` | `oklch(0.708 0 0)` | Secondary text |
| **Accent** | `oklch(0.97 0 0)` | `oklch(0.9851 0 0 / 5%)` | Hover, subtle highlight |
| **Border** | `oklch(0.922 0 0)` | `oklch(1 0 0 / 10%)` | Dividers, outlines |
| **Success** | `oklch(0.7775 0.2447 144.9)` | same | Positive, returns, green |
| **Warning** | `oklch(0.769 0.188 70.08)` | same | Alerts, amber |
| **Destructive** | `oklch(0.577 0.245 27.325)` | `oklch(0.5961 0.2006 36.48)` | Errors, danger |

**Fallback (pre-Tailwind):** Background `#1a1a1d`, text `#fafafa`.

**Approximate hex (dark mode):**
- Background: ~`#1f1d1e`–`#1a1a1d`
- Foreground: ~`#fafafa`
- Primary: ~`#6d66e8` (violet/indigo, hue 263°)

### Typography (Product / App)

Defined in `src/frontend/index.css` and Tailwind theme.

| Use | Font | Fallback | Notes |
|-----|------|----------|--------|
| **Display / Headlines** | **Rebels** | sans-serif | Custom: `/Rebels-Fett.woff2`, `.woff` — `font-display` in UI |
| **Body / UI / Code** | **Roboto Mono** | monospace | Default body and data; `font-mono` for labels, stats, code |

- **Rebels** gives a sharp, technical, cinematic headline feel.
- **Roboto Mono** keeps body and data clear and consistent.
